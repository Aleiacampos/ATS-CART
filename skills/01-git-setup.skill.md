# Skill: Git Setup e Commit Inicial

## Objetivo
Configurar um repositório Git local para o projeto do dashboard CAR-T e criar o commit inicial.

## Ações suportadas
- iniciar um repositório Git (`git init`)
- configurar nome e e-mail localmente (`git config user.name`, `git config user.email`)
- criar `.gitignore`
- adicionar arquivos essenciais (`index.html`, `README.md`)
- realizar o primeiro commit no branch `main`

## Uso
1. Verifique se o diretório não é um repositório Git.
2. Inicialize o repositório.
3. Configure identidade local se necessário.
4. Adicione e comite os arquivos.

## Resultados esperados
- `.git/` criado
- `main` como branch padrão
- commit inicial com os arquivos de deploy do dashboard
