# Skills do projeto ATS-CART

Este diretório contém skills documentadas para o fluxo completo do dashboard CAR-T.

## Skills incluídas
- `01-git-setup.skill.md` — inicialização do repositório Git e commit inicial
- `02-github-remote.skill.md` — conexão com o remoto GitHub e push para `ATS-CART`
- `03-github-pages.skill.md` — validação e ativação do GitHub Pages

## Como usar
Cada arquivo descreve a ação, os comandos necessários e os resultados esperados.

## Contexto
O projeto inclui um `index.html` protegido por senha, pronto para deploy no GitHub Pages.
