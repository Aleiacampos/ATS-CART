# Skill: Configurar remoto GitHub e enviar para o repositório

## Objetivo
Conectar o repositório local a um remoto no GitHub e enviar o branch `main`.

## Ações suportadas
- adicionar remote `origin` com URL GitHub
- atualizar a URL do remote caso já exista
- enviar o branch `main` para o remoto usando `git push -u origin main`

## Uso
1. Confirme a URL do repositório GitHub.
2. Se necessário, ajuste o remote existente.
3. Faça push para o GitHub.

## Resultado esperado
- `origin` apontando para `https://github.com/Aleiacampos/ATS-CART.git`
- branch `main` disponível no GitHub
- tracking de `origin/main` configurado
